# Aroma Cafe ☕

Responsive Coffee Shop Ecommerce Website built using HTML, CSS, Bootstrap 5 and JavaScript.

---

## Features

- User Login & Registration
- LocalStorage Authentication
- Wishlist Functionality
- Add To Cart System
- Quantity Increase/Decrease
- Coupon Discount System
- Responsive Bootstrap 5 Design
- Modern Ecommerce UI

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- LocalStorage

---

## Pages

- Home Page
- Products Page
- Wishlist Page
- Cart Page
- Login Page
- Register Page
- Checkout Page
